This

Uso do this para acessar os atributos dentro da classe, quando se tem o uso do Construtor
Ex:
class Carro{
    private String modelo;
    private int ano;
}
Carro(String m, int a){
    this.modelo = m;
    this.ano = a;
}
String mostrar(){
    return modelo + ano;
}

Na instancia da classe fica:
-> Carro c1 = new Carro("polo", 2025);
